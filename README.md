# Elysium Rules

Официальная страница правил проекта **Elysium**.

На сайте размещены правила для нескольких направлений проекта:

- **Elysium Java**
- **Elysium Bedrock**
- **Telegram / Discord чаты**

Страница создана для удобного просмотра правил игроками и участниками сообщества.

## Сайт

Страница опубликована через GitHub Pages.

## Разделы

### Java

Правила Java-направления: честная игра, стороннее ПО, поведение игроков, администрация, производительность сервера, баги и дюпы.

### Bedrock

Правила Bedrock-направления: IP/CID, команды, телепортации, чат, вред серверу и общие положения.

### Telegram / Discord

Общие правила общения в чатах проекта: оскорбления, медиа-контент, флуд, оффтоп, реклама, дезинформация и взаимодействие с администрацией.
