# Elysium Minecraft

Официальный сайт проекта **Elysium Minecraft**.

Сайт создан как основная страница проекта, где собрана важная информация для игроков и участников сообщества.

## Сайт

Официальная страница:

https://wandraker.github.io/Elysium-Minecraft/

## Разделы сайта

На данный момент на сайте доступны:

- **Elysium Java**
- **Elysium Bedrock**
- **Telegram / Discord чаты**

В будущем сайт может быть расширен: главная страница проекта, онлайн-карта, информация о подключении, новости, ссылки на сообщество и другие разделы.

## Elysium Java

Раздел Java-направления проекта.

Содержит правила, связанные с честной игрой, сторонним ПО, поведением игроков, администрацией, производительностью сервера, багами и дюпами.

## Elysium Bedrock

Раздел Bedrock-направления проекта.

Содержит правила, связанные с IP/CID, командами, телепортациями, чатом, вредом серверу и общими положениями.

## Telegram / Discord

Раздел правил общения в чатах проекта.

Содержит правила по оскорблениям, медиа-контенту, флуду, оффтопу, рекламе, дезинформации и взаимодействию с администрацией.
