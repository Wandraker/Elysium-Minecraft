Elysium Minecraft Server — v2.7.0

Структура проекта:
- index.html
- css/style.css
- js/app.js

Что изменилось:
- CSS остаётся в отдельном css/style.css;
- весь JavaScript, роутинг, правила, страницы, эффекты и встроенные WebP-аватарки
  перенесены из index.html в js/app.js;
- внешний вид и функции должны остаться такими же, как в v2.6.0 / v2.5.1;
- новых визуальных функций в этой версии не добавлялось.

Как обновить GitHub Pages:
1. Распакуйте архив.
2. В корень репозитория загрузите:
   - index.html
   - папку css
   - папку js
3. Старый index.html замените новым.
4. Папку css можно заменить целиком.
5. Папку js нужно загрузить обязательно.
6. README.txt на работу сайта не влияет.
7. После загрузки при необходимости очистите кэш Cloudflare.

Итоговая структура репозитория:
index.html
css/
  style.css
js/
  app.js

Важно:
- index.html нельзя открывать отдельно без папок css и js;
- при локальном открытии Android через content:// внешние CSS/JS могут блокироваться;
- для проверки используйте GitHub Pages или локальный HTTP-сервер.
